# User Stories - FirstMod-AI

This document contains user stories for all four actor types in the FirstMod-AI application: Guest, User, Subscriber, and Admin.

---

## 1. GUEST (Unauthenticated User)

**US-G-001** - As a guest, I want to sign up for an account so that I can access the platform's features.

**US-G-002** - As a guest, I want to view sample animations on the homepage so that I can see what the platform can do.

**US-G-003** - As a guest, I want to log in with my credentials so that I can access my account.

**US-G-004** - As a guest, I want to see the FAQ page so that I can get answers to common questions.

**US-G-005** - As a guest, I want to see the pricing page so that I can understand subscription plans.

## 2. USER (Free Authenticated User)

### 2.1 Authentication

**US-U-001** - As a user, I want to log in to my account so that I can access the platform.

**US-U-002** - As a user, I want to log out of my account so that I can securely end my session.

### 2.2 Face Swap Feature

**US-U-003** - As a user, I want to use the free Face Swap feature so that I can create face swap images.

**US-U-004** - As a user, I want to choose a source image for face swap so that I can select which face to use.

**US-U-005** - As a user, I want to choose a target image for face swap so that I can specify where the face should appear.

**US-U-006** - As a user, I want to save my face swap result to my dashboard so that I can access it later.

**US-U-007** - As a user, I want to download my face swapped image so that I can use it outside the platform.

**US-U-013** - As a user, I want to preview the source image for face swap so that I can verify it before processing.

**US-U-014** - As a user, I want to preview the target image for face swap so that I can verify it before processing.

**US-U-015** - As a user, I want to see a progress bar for face swap processing so that I know how long it will take.

**US-U-016** - As a user, I want to see the connection status of the face swap API so that I know if the service is available.

**US-U-017** - As a user, I want to preview the face swap result so that I can review it before saving.

### 2.3 Profile Management

**US-U-008** - As a user, I want to edit my profile picture so that I can personalize my account.

**US-U-009** - As a user, I want to update my profile information (name, email) so that I can keep my account details current.

**US-U-010** - As a user, I want to change my password so that I can maintain account security.

**US-U-011** - As a user, I want to delete my account so that I can permanently remove my data from the platform.

### 2.4 Subscription

**US-U-012** - As a user, I want to pay for a subscription so that I can access premium features.
